# 仿微信聊天室项目-前端

## 项目描述

这是一个基于 `vue3 + element-plus + vite3` 的仿微信聊天室项目。

## 开发环境

* node版本：16.14.2
* 浏览器：chrome 99.0.4844.51
* vscode版本：1.66.2

## 项目初始化和构建流程

### 1、依赖包安装

```shell
npm install
```

### 2、运行开发服

```shell
npm run dev
```

### 3、项目构建

```shell
npm run build
```

### 4、文件 eslint 检测并修复文件 eslint 报错

```shell
npm run lint:fix
```
